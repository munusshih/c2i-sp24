  • Flexbox Letterforms

    Create a new subfolder called exercise-3 within your username.github.io folder. Create a new index.html and styles.css file within that folder to work in. Explore the use of Flexbox to create gridded letterforms. You can choose a word, create an alphabet, or go crazy on one letter. If you want to lean into pixelation, great. If you want to create more organic letterforms, consider what other CSS properties you can use to break up the rigidity of the grid. Maybe border-radius? Softer colors? Shapes that are not just squares and circles? Play around!
