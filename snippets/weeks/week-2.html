  • HTML Recipe

    Let’s refresh our memory of HTML and practice web typography fundamentals by typesetting a recipe as a webpage! First, choose an existing recipe you’re interested in working with (e.g. Fluffy Jiggly Japanese Cheesecake). You’ll then use only HTML to typeset that recipe.

    Part 1: Semantic HTML

    Create a new subfolder called exercise-1a within your username.github.io folder. Create a new index.html within that subfolder to work in. Take a look at your chosen recipe and note the different levels of hierarchy you’ll need. Use semantic HTML such as headings (h1 through h6), ordered lists (ol), and/or unordered lists (ul) to create hierarchy and structure for your content.

    Part 2: HTML as Design

    Now create a new subfolder called exercise-1b within your username.github.io folder. Create a new index.html within that subfolder to work in. Using the same recipe, use only one HTML element to design it. See how far you can push your design using just that one HTML element: how can hierarchy still be achieved? Can the element be used illustratively? Can the HTML element relate to the contents of the recipe in some way? Get experimental, get weird, get creative!

    For both exercises, please include a link to the original recipe you’ve chosen as an HTML comment.
