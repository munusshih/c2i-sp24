  • Random Poem Generator

    Create a new subfolder called exercise-4 within your username.github.io folder. Create a new index.html, styles.css, and script.js file within that folder to work in. Put your JavaScript skills into practice by creating a random poem generator that produces a unique poem each time the user refreshes the page. The general structure of the poem can remain consistent, but certain words are randomized upon refresh. Consider which words should be randomized (e.g. nouns, adjectives, and/or verbs) and how doing so can drastically change the meaning and tone of your poem. Your poem should be responsive and styled with CSS.

    Reference the JavaScript Basics Worksheet you completed for additional reinforcement!

    2. This exercise is linked out from my class homepage.

    3. I used JavaScript to make my generator produce a new and unique poem upon refresh.

    4. I styled my poem with CSS.

    5. My poem is responsive.
